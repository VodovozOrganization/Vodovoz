.net standart библиотека с сущностями
их можно использовать во всех проектах, включая службы в докере

Если полностью перенести сущность на данный момент нет возможности, то
в старой библиотеке остается сущность с тем же названием как и была, а
в эту библиотеку переносятся все свойства какие точно можно перенести.
Класс в старой библиотеке наледует класс от текущей библиотки.
Чтобы не было конфликта имен, новый класс называем <КЛАСС>Entity

Маппинг так же наследуется.
